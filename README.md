# Java Socket聊天室

#### 介绍
（个人学习用）一个用Java多线程，集合，Socket编程，面向对象等内容编写的简易聊天室工程，后续会进行一些内容的补充、改进和完善。
本项目主要是使用到了Java基础知识部分，可以用来巩固所学到的Java多线程、集合、IO编程、Socket编程，以及重要的面向对象编程。
